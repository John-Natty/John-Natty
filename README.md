## À propos de moi

Je suis actuellement en formation en développement web chez **Holberton School Toulouse**.

J’apprends à construire des projets concrets, à comprendre la logique derrière le code et à développer des solutions utiles, propres et maintenables.

Ce qui me motive dans le développement, c’est de transformer une idée en projet fonctionnel : une interface, une base de données, une fonctionnalité ou une application pensée pour l’utilisateur.
